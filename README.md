Проект: Second-Hand Market (SPA)

Данное приложение является многостраничным веб-интерфейсом для площадки секонд-хенд товаров. Проект выполнен в рамках стажировки с целью демонстрации навыков работы с React, Redux Toolkit и ручной настройки сборки проекта.

Краткая функциональность

Авторизация: Реализован полный цикл входа в систему с использованием JWT-токенов через API. 

Маршрутизация: Настроена навигация между тремя типами страниц: страница логина, главная страница (каталог) и детальная страница товара.

Управление состоянием: Глобальное состояние приложения (авторизация, фильтры, корзина, список избранного) управляется с помощью Redux Toolkit.

Работа с API: Получение, фильтрация и кэширование данных о товарах реализовано через RTK Query на базе DummyJSON API.

Фильтрация и поиск: Реализован поиск товаров в реальном времени, фильтрация по категориям и брендам, а также сортировка по цене.

Корзина и избранное: Реализована логика добавления товаров в корзину и управления списком избранного с синхронизацией состояния в хедере приложения.

Адаптивная верстка: Интерфейс корректно отображается на десктопных и мобильных устройствах. Реализовано мобильное меню (бургер).

Сборка: Проект собран с нуля с использованием Webpack без применения готовых решений.

Список зависимостей
Основные зависимости (Dependencies):

react (v18+) — библиотека для создания пользовательских интерфейсов.

react-dom — пакет для работы с DOM.

react-router-dom (v6) — библиотека для организации навигации.

@reduxjs/toolkit — инструмент для эффективной работы с Redux.

react-redux — официальная библиотека для интеграции Redux и React.

Зависимости для разработки (Dev Dependencies):

webpack, webpack-cli, webpack-dev-server — сборщик проекта и сервер для разработки.

babel-loader, @babel/core, @babel/preset-env, @babel/preset-react — инструменты для транспиляции современного JavaScript и JSX кода.

css-loader, style-loader — лоадеры для обработки и подключения CSS-файлов.

html-webpack-plugin — плагин для генерации HTML-файла на основе шаблона.

file-loader — инструмент для обработки изображений и других медиафайлов.

Инструкция по запуску

Для запуска проекта на локальной машине выполните следующие команды:

Клонирование репозитория:

git clone https://github.com/etoneyaaa402/second-hand-market
cd second-hand-market

Установка всех необходимых пакетов:

npm install

Запуск приложения в режиме разработки:

npm start

После запуска приложение будет доступно по адресу http://localhost:3000.

Сборка проекта для продакшена:

npm run build

Скомпилированные файлы будут находиться в директории dist/.

Ссылка на деплой: https://second-hand-market-one.vercel.app/
